### 적용 기능
- Netlify를 통한 자동화 배포
- Open Graph & Twitter Cards를 통한 정보 지정
- Google Fonts를 사용한 동일한 서체 적용
- GSAP & ScrollToPlugin를 통한 애니메이션
- Swiper를 통한 슬라이드 기능
- Youtube API를 통한 동영상 제어
- ScrollMagic을 통한 스크롤 기능
